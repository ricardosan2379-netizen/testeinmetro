Cédulas do Real — elementos de segurança lidos
  • Número de série: sequência alfanumérica única por cédula (ex: A1234567-8)
  • Data Matrix: código 2D compacto presente em cédulas mais recentes
  • Code 128 / Code 39: barcode linear com dados da série
  • QR Code: em algumas séries, link para verificação BCB
  • ITF / EAN: possíveis em embalagens de pacotes
Modo Foto / Upload
  • Tire uma foto diretamente com a câmera do dispositivo
  • Ou faça upload de foto já tirada
  • A imagem é processada 100% local — sem envio para servidores
  • Melhor resultado: boa iluminação, foco, cédula plana
  • Tente tanto a frente quanto o verso da cédula
Objetivo do protótipo
  • Descobrir quais formatos o leitor consegue detectar
  • Identificar qual campo contém o número único da cédula
  • Exportar o log para análise e decisão de integração
Dicas de leitura
  • Use Câmera ao Vivo para leitura rápida em tempo real
  • Use Foto/Upload quando a leitura ao vivo não funcionar
  • Na foto, aproxime bem e mantenha foco no código específico
  • O verso das cédulas recentes costuma ter mais elementos de máquina
