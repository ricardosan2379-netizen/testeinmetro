Contexto técnico
Por que o QR Code é ignorado?
  • O QR Code do selo INMETRO (Portaria 314/2025) contém apenas o link para baixar o app oficial
  • Exemplo do conteúdo: https://napalmadamao.inmetro.gov.br/
  • O UUID do produto aparece na mensagem WhatsApp gerada pelo app oficial
O que este app faz
  • Lê todos os formatos exceto QR Code
  • Captura o texto bruto completo de cada código
  • Analisa padrão: numérico, alfanumérico, UUID, URL, EAN
  • Registra comprimento, encoding e formato detectado
  • Salva um log completo exportável em JSON
Próximo passo
  • Escaneie selos reais e exporte o log
  • Analise quais formatos o leitor consegue ler
  • Identifique qual código contém o identificador único do produto
  • Integre ao PCI Manager usando esse campo específico
